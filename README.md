# Url Shortening Service Redis/Sinatra

Simple overview of use/purpose.

## Description

A simple URL shortening service with additional JSON API.

## Getting Started

### Dependencies

* You will need Ruby, Bundler, and Redis installed.

### Installing

* Clone this Repo
* bundle install

### Executing program

* 
```sh
rackup config.ru
```
